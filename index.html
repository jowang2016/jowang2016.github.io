mac搭建hexo+github博客

安装node之后，通过npm安装hexo
‘$ npm install -g hexo’
创建hexo项目，例如你想创建的项目名称为’myblog’：
‘hexo init myblog’
创建成功之后，进入项目目录:
‘cd myblog’
通过’ls’命令可以查看目录的结构
此时想要写文章，最好借助ide，我选择的是sublime
在sublime中通过 project–add folder to project 将myblog导入到sublime
点开myblog下面的source文件夹，可以看到_posts文件夹，所有的博客文章都在这个目录下面
现在你可以尝试编辑一篇文章
在将你的博客部署到github之前，你可能想看一下此时的博客效果，这个并不难，只要在终端，hexo项目目录下输入：
‘hexo server’
便可以通过打开http://localhost:4000/ 查看博客效果
下面准备将博客发布到Github上，步骤如下：
依然在终端myblog目录下，输入：
‘vi _config.yml’
编辑这个文件的末尾处：
‘deploy:
type: git
repo: git@github.com:xxxx/xxxx.github.io.git
branch: master’
xxxx为你的github账户名称
所以你需要在自己的github上创建名称为xxxx.github.io 的仓库
创建好之后，输入如下命令：
‘$ npm install hexo-deployer-git –save’
‘hexo generate’
‘hexo deploy’
ok！现在是见证奇迹的时刻了！
在浏览器中输入https://xxxx.github.io （xxxx为你的github账户名称）
看看效果吧！
之后提交的命令：
hexo clean #清除缓存
hexo generate
hexo deploy
